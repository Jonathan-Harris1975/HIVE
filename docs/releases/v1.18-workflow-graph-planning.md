# HIVE v1.18 – Workflow Graph Planning

Build path: consolidated into `v1.22-workflow-simulation-persistence`.

v1.18 adds a graph model for HIVE workflow planning. It converts skill recommendations and shared execution plans into nodes and edges that a future UI can render.

## Added

- Workflow graph templates.
- Graph-shaped plan generation.
- Node status metadata.
- Sequential edges for UI display.
- Free-tier-safe synchronous graph building.

## Safety

This release is planning-only. It does not execute skills, mutate repositories, install packages, create background jobs or write external exports.

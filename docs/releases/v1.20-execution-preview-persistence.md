# HIVE v1.20 – Execution Preview Persistence

Build path: consolidated into `v1.22-workflow-simulation-persistence`.

## Added

- `POST /v1/execution-preview/save`
- `GET /v1/execution-preview/history`
- `GET /v1/execution-preview/{preview_id}`

## Safety

Execution previews are stored in the D1 `execution_previews` lane only. The endpoint does not execute skills, mutate repos, install packages, write R2 exports, or start background jobs.

# HIVE v1.19 – Controlled Execution Preview

Build marker: `v1.22-workflow-simulation-persistence`.

v1.19 adds controlled execution preview semantics on top of workflow graphs. HIVE can now show what *would* need to happen before execution while still refusing to execute anything directly.

## Added

- `GET /v1/workflow-graphs/templates`
- `POST /v1/workflow-graphs/build`
- `GET /v1/execution-preview/policies`
- `POST /v1/execution-preview`
- Step statuses for graph nodes.
- Blocker metadata for review gates and disabled adapters.
- Next-required-actions list for future UI and MAST hand-off.

## Safety policy

v1.19 remains non-executing:

- No skill execution.
- No repo mutation.
- No package install.
- No R2 export write.
- No background jobs.
- No adapter execution.

Even if a plan is approved, `can_execute_now` remains `false` until a future explicit adapter layer is implemented and reviewed.

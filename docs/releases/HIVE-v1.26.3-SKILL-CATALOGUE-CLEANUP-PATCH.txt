HIVE v1.26.3 - Skill Catalogue Cleanup Patch

Purpose
- Adds a safe D1-only cleanup path for legacy bad/duplicate skill records that were created when ordinary uploaded files were incorrectly registered as skills.
- R2 objects are not deleted or modified by this cleanup.

Backend changes
- Adds POST /v1/skills/cleanup-uploaded-file-skills.
- Dry-run is the default.
- Live cleanup requires confirm='delete-uploaded-file-skills'.
- Keeps reviewed descriptor skills created from hive_skills lane under skills/.
- Adds D1MetadataStore.delete_metadata_ids for bounded per-row deletion reporting.

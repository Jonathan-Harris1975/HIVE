# HIVE v1.16 – Skill Search Review Integration

## Purpose

v1.16 consolidates the previously split skill-search and execution-review work into one clean forward build line. The repo had drifted between `v1.9-intelligent-skill-search` and later review/evidence-pack features. This release repairs that drift and makes the build deployable again.

## Included layers

- Weighted skill search over the imported D1 skill catalogue.
- Skill lookup helpers: by id, repo, risk and lane.
- Skill recommendation endpoint.
- Review-gated skill routing endpoint.
- Shared ecosystem execution-plan endpoint.
- Execution review queue.
- Review decisions.
- Audit trail and evidence-pack endpoints.
- Repo hygiene scan endpoint.

## Safety model

This release remains plan-only and review-gated. HIVE does not install skills, mutate repos, write evidence exports to R2, or start background execution jobs from these endpoints. Approved review items still return `can_execute_now: false`.
